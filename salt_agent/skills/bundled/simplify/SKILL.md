# Simplify

Review the recently changed code for:
1. Unnecessary complexity — can anything be made simpler?
2. Dead code — are there unused functions, variables, imports?
3. Repeated patterns — can anything be extracted?
4. Over-engineering — is there complexity that isn't needed yet?

Read the changed files, identify issues, and fix them.
